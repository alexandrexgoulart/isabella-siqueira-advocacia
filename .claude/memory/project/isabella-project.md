# Projeto Isabella Siqueira Advocacia

## Descrição
Este é um website profissional para o escritório de advocacia de Isabella Siqueira, especializado em Direito Previdenciário no Brasil. O site apresenta uma estética sofisticada com foco em converter visitantes em clientes através de uma apresentação profissional dos serviços jurídicos.

## Tecnologias
- HTML5
- CSS3 (com animações e transições)
- JavaScript (funcionalidades interativas e painel administrativo)
- Font Awesome (ícones)
- Google Fonts (tipografia)

## Funcionalidades Principais
- Design responsivo completo
- Seção de serviços organizada em abas
- Painel administrativo para edição de conteúdo
- Otimização de performance
- Recursos de acessibilidade implementados
- Boa implementação de SEO
- Sistema de vídeo explicativo sobre restituição
- Formulário de contato integrado com WhatsApp

## Arquitetura
- Single Page Application (SPA) com rolagem suave para seções
- Estrutura modular com componentes reutilizáveis
- Sistema de grid personalizado para layout
- Animações e transições CSS avançadas
- Painel administrativo integrado no próprio frontend

## Estado Atual
O projeto está funcional e publicado, com um painel administrativo interno que permite a edição de conteúdo. O site já passou por várias iterações e otimizações de segurança e performance.

## Objetivo
Fornecer um site profissional para o escritório de advocacia de Isabella Siqueira, ajudando a converter visitantes em clientes através de uma apresentação profissional dos serviços jurídicos.
