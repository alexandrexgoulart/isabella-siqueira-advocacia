# Isabella Siqueira Advocacia — Site Institucional + Blog

Site profissional de advocacia especializada em **Direito Previdenciário**, desenvolvido como single-page application com HTML, CSS e JavaScript puros, incluindo blog completo para artigos.

🔗 **[Ver site ao vivo](https://alexandrexgoulart.github.io/isabella-siqueira-advocacia/)**

---

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas (sem framework)
- JavaScript vanilla (sem dependências)
- GitHub Pages (hospedagem gratuita)
- Supabase (banco de dados para blog)
- Imagens `.avif` para máxima performance


## Funcionalidades

### Site Principal
- Navegação responsiva com menu mobile
- Sistema de abas para serviços previdenciários
- Vídeo explicativo otimizado
- Formulário de contato integrado
- Banner e modal de LGPD/cookies
- Modo claro forçado (`color-scheme: light only`)
- Acessibilidade WCAG AA

### Blog
- Listagem de artigos com busca e filtro
- 30 categorias (todas as áreas de atuação)
- Paginação com botão "Ver Mais"
- Página individual com formatação
- Admin completo para gerenciar artigos
- Upload de imagens
- Toolbar de formatação + 30 emojis
