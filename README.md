# Isabella Siqueira Advocacia — Site Institucional

Site profissional de advocacia especializada em **Direito Previdenciário**, desenvolvido como single-page application com HTML, CSS e JavaScript puros.

🔗 **[Ver site ao vivo](https://alexandrexgoulart.github.io/isabella-siqueira-advocacia/)**

---

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas (sem framework)
- JavaScript vanilla (sem dependências)
- GitHub Pages (hospedagem gratuita)
- Imagens `.avif` para máxima performance

## Paleta de Cores — Navy & Gold

| Cor | Hex | Uso |
|-----|-----|-----|
| Navy | `#1A2B4A` | Primary — autoridade jurídica |
| Gold | `#C5964B` | Accent — sofisticação |
| Ivory | `#F8F6F1` | Background — acessibilidade |
| Green | `#059669` | WhatsApp / sucesso |

## Estrutura de Arquivos

```
├── index.html              # Site completo (single-page)
├── styles.css              # Todos os estilos
├── script.js               # JavaScript principal
├── foto_sobre.avif         # Foto da advogada
├── isabella_sobre.avif     # Foto hero
├── optimized_video/        # Vídeo otimizado (4.3MB)
└── docs/                   # Documentação técnica
```

## Funcionalidades

- Navegação responsiva com menu mobile
- Sistema de abas para serviços previdenciários
- Vídeo explicativo otimizado
- Formulário de contato integrado
- Banner e modal de LGPD/cookies
- Modo claro forçado (`color-scheme: light only`)
- Acessibilidade WCAG AA

## Áreas de Atuação

- Aposentadoria por Tempo de Contribuição e Idade
- Auxílio Doença e Auxílio Acidente
- Pensão por Morte
- BPC/LOAS
- Revisão de Benefícios
- Planejamento Previdenciário

## Deploy

O site é publicado automaticamente via GitHub Pages a cada push na branch `main`.
